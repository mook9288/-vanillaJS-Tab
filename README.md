# vanillaJS-Accordion
- Making Accordion with Vanilla Coding


## 아코디언 만들기

1. 아코디언의 기본 요소
  - 아코디언 헤더
  - 아코디언 컨텐츠

2. 아코디언의 기본 조건
  - 헤더와 컨텐츠의 1:1 맵핑

3. 아코디언의 기능
  - 아코디언 활성화 > 아코디언 컨텐츠 활성화 (Show)
  - 나머지 아코디언 비활성화 > 컨텐츠 비활성화 (Hide)

4. 추가 기능
  - url Hash 여부
  - 부모의 자식 안에 또 다른 아코디언이 있을 경우
  - 토글 여부 1:n // n:n
  - 에니메이션
