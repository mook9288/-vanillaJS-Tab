### 큐브 조작 방법
#### 회전 방향
- 시계방향: `'`이 없이 알파벳만 있는 경우
- 반시계방향: `'`이 알파벳 뒤에 존재하는 경우 (예) `F'`)

#### 회전 위치
- `U`, `U'` – 상 (Up)
- `D`, `D'` – 하 (Down)
- `L`, `L'` – 좌 (Left)
- `R`, `R'` – 우 (Right)
- `F`, `F'` – 앞 (Front)
- `B`, `B'` – 뒤 (Back)
