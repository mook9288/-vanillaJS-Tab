# RubiksCube
루빅스 큐브 구현

## Step-1
### 구현단계
- [x] 마크업 구조
  - 단어, 정수(밀어내고 싶은 수), 밀어낼 방향 입력창
  - 결과값 확인 버튼
- [x] input에 입력된 값이 없을 경우, 입력하라는 경고창 띄운다.
- [x] 정수는 -100보다 크거나 같고 100보다 작아야한다.( -100 <= N < 100)
  - 정수보다 단어의 길이가 클 수 있다.
  - 단어의 길이의 배수로 정수를 나누고, 나머지 값으로 단어를 밀어낸다.
- [x] 밀어낼 방향은 우는 R, 좌는 L로 나타낸다.
  - 대소문자 모두 입력 가능하다.
