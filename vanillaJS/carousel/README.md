# Carousel UI
- 이미지 슬라이더 만들기
![Carousel UI](images/img.png)
위에 주어진 이미지와 같이 생긴 화면 제작

### 요구사항
- 페이지 로딩 시, 첫 번째 이미지 화면으로 시작
- 좌측 화살표 클릭시 이전 이미지
- 우측 화살표 클릭시 다음 이미지
- 5번째 이미지에서 우측 화살표를 누를 경우, 1번째 이미지
- 1번째 이미지에서 좌측 화살표를 누를 경우, 5번째 이미지
- Dot는 이미지 수만큼 노출
- Dot를 누를 경우, 해당 순번의 이미지

### 참고사항
- 이미지 변경은 이미지 태그의 src를 변경하는 방식
